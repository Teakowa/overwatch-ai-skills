# Skills Catalog

Skill folders in this catalog are authored against the [Agent Skills Specification](https://agentskills.io/specification) and intended to be portable across Codex, Claude Code, and other compatible runtimes.

| Name | Purpose | Path | Default prompt summary |
| --- | --- | --- | --- |
| `ow-workshop-loops` | Design, review, and optimize Overwatch Workshop loop safety/performance. | `skills/ow-workshop-loops/` | Use `$ow-workshop-loops` to design or review loop logic with safe waits, loop selection, and server-load safeguards. |
| `ow-workshop-server-stability` | Audit and remediate Workshop server-load crash/stability risks. | `skills/ow-workshop-server-stability/` | Use `$ow-workshop-server-stability` to diagnose startup crashes, spikes, and event/loop overload, then apply targeted safe fixes. |
